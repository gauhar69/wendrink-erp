# Опасные скрипты

Эти скрипты изменяют данные в продакшн БД. Перед запуском:
1. Сделать бэкап: `cp wendrink.db wendrink.db.before-script-$(date +%Y%m%d-%H%M)`
2. Прочитать что скрипт делает
3. Уведомить Армана

Запуск только с явным указанием БД:
```
python scripts/dangerous/delete_day.py 2026-04-09
```
